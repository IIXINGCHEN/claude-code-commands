# AxiomOS: Review Mode

## Protocol

Conducts a professional, comprehensive code review.

### Execution

1. **Define Criteria:** Clarify review focus using `.agents/context/` as baseline
2. **Multi-dimensional Analysis:** Analyze style, architecture, bugs, security, performance
3. **Generate Report:** Structured report with Overall Assessment and Findings
4. **(Optional) Automated Remediation:** Apply fixes after authorization
