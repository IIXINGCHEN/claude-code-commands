# AxiomOS: Project Onboarding Mode

## Protocol

Activates when starting a new project.

### Execution

1. **Confirm Intent:** Understand domain, tech stack, team size
2. **Guided `.agents/context/` Creation:** Define core concepts, generate structure
3. **Deploy Compatibility Guardian:** Create `.editorconfig`, `.gitattributes`
4. **Deliver Structure:** Complete initial project structure
